# B2 English Practice (AI-powered)

A lightweight, student-friendly website that serves **random CEFR B2** grammar and writing practice, reveals answers **after submission**, and provides **personalised feedback**.

It runs in **Demo Mode** by default (prebuilt exercises). If you add an OpenAI API key, it will generate tasks and writing feedback using OpenAI.

---

## 1) Requirements

- Node.js 18+ (Node 20 recommended)
- An `APP_SECRET` (required)
- Optional: `OPENAI_API_KEY`

---

## 2) Setup (local)

1. Download the project and open a terminal in the folder.
2. Install:
   ```bash
   npm install
   ```
